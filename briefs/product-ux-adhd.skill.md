# # Skill: Product-UX-ADHD 

# ## Purpose 

Design, review, and improve product screens and flows (landing page, signup, upload, player, settings) so they are calm, clear, and ADHD-friendly. 

# ## When to use 

- Working on the website (landing, signup, upload, player). 

- Judging whether a UI is overwhelming or confusing. 

- Planning changes to flows that affect ADHD and neurodivergent users. 

# ## Inputs 

- Current screen or flow description (text, components, rough wireframe). 

- Optional: target audience and goal for that screen. 

- Optional: constraints (must keep X, cannot change Y). 

# ## Core principles 

- Design for: 

- working memory limits, 

- time blindness, 

- task initiation difficulty, 

- distractibility, 

- sensory overload, 

- rejection sensitivity, 

- executive function load. 

- Prefer one primary action per screen. 

- Make the next step obvious and visible. 

- Explain waits, states, and trade-offs. 

- Reduce shame and increase self-efficacy. 

- Keep layout calm and visually prioritized. 

# ## Behavior 

When this skill is active: 

1. **Assess clarity and overload.** 

- Identify confusing sections. 

- Identify places with too many choices. 

- Identify missing feedback or progress cues. 

2. **Explain what’s working.** 

- Point out elements that help ADHD users (clear steps, simple language, visible next actions). 

# 3. **Recommend changes.** 

- Suggest 3–7 concrete changes, each small and actionable. 

- Prioritize changes that reduce cognitive load and increase clarity. 

4. **Output format.** 

- Short summary (2–3 sentences). 

- Two lists: 

- "Strong elements" (bullets). 

- "Changes to make next" (numbered list). 

# ## Example prompt 

Mode: Product-UX-ADHD 

Task: Review my landing page hero for ADHD clarity. 

Screen: 

- Headline: "FocusReader – Transform your reading." 

- Subheadline: "Upload your PDFs and listen instead of reading." 

- CTAs: "Get started" and "Learn more". 

Instructions: 

- Identify what’s confusing or overwhelming. 

- Identify what’s clear. 

- Give me 3 concrete changes that are ADHD-friendly. 

# ## Success criteria 

- The answer makes it easier to see what to change first. 

- Recommendations are concrete and small enough to do in one session. 

- The revised screen feels calmer, clearer, and more usable to ADHD and neurodivergent users.
